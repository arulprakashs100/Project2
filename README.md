# 🚀 Full Stack Login & Registration System

This project is my first complete full-stack application where I implemented a working authentication system by connecting frontend, backend, and database.

---

## 🧩 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot (Java)
- REST APIs

### Database
- PostgreSQL
- JPA / Hibernate

---

## 📌 Features

- User Registration
- User Login Authentication
- Form Validation
- Error Handling
- Data stored in PostgreSQL
- Frontend ↔ Backend integration using Fetch API

---

## 🔄 Application Flow

1. User enters email & password in frontend (HTML form)
2. JavaScript sends request to backend using Fetch API
3. Spring Boot backend receives request via REST Controller
4. Service layer processes login/register logic
5. Repository interacts with PostgreSQL database
6. Response is sent back to frontend
7. User is redirected based on result

---

## 🧠 Key Learnings

- How frontend communicates with backend using APIs
- Structuring backend using Spring Boot architecture
- Connecting and performing operations with PostgreSQL
- Understanding full request-response lifecycle
- Debugging real integration issues
